# incoming/

Landing zone for **standalone HTML files** generated by an LLM
(Claude Design, Claude Cowork, ChatGPT, etc.) that don't yet fit into the
connector/art structure.

## Workflow

1. Drop the HTML file here — name it something descriptive like
   `2026-04-23-spiral-strip.html`.
2. Open `playtable.html?incoming=2026-04-23-spiral-strip.html` to view
   it in an iframe alongside a "back to play table" link.

You can look at the incoming design without disturbing the live viewer.

## Promoting to a real variant

If you like an incoming design and want to keep it:

- If it's a **strip** style → extract the render function into a new
  `connectors/<name>.js` using the pattern in `connectors/README.md`.
- If it's an **art** style → same idea in `art/<name>.js`.
- Then delete the file from `incoming/`.

Don't keep designs here long-term — this folder is a scratch area, not a
library.
